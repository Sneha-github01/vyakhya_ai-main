## Deployment

To deploy this project run

```bash
  flutter run
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/suraj-yadav0/vyakhya_ai-main
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter  pub get
```

Start the server

```bash
  flutter run
```
